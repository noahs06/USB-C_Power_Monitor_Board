# USB-C_Power_Monitor_Board
STM32 based board displays how much power your hardware draws over USB-C. Displays to LCD1602 via I2C. Includes custom library to initialize and output to LCD. Utilizes STM32 NUCLEO-F302R8, LCD Driven by PCF8574T (https://www.ti.com/lit/ds/symlink/pcf8574.pdf?ts=1627006546204)
